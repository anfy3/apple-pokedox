\# Apple-Inspired Pokédex



A minimalist Pokédex web application inspired by Apple’s design language.



\## Features

\- Pokémon search using PokeAPI

\- Apple-style UI

\- Animated stat bars

\- Shiny Pokémon toggle

\- Dynamic theme based on Pokémon type



\## Tech Stack

\- HTML

\- CSS

\- JavaScript

\- PokeAPI
